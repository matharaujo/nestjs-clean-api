# NestJS API

### Configure Environment
 - [Production](.env)
 - [Development](.env.development)


### Install Dependencies
```bash
$ npm install

# husky
$ npm run husky:install
```

### Running the app
```bash
# development mode
$ npm run start

# production mode
$ npm run start:production

# build
$ npm run build
```

### Test
```bash
# unit tests
$ npm run test

# tests watch mode
$ npm run test:watch
```

### Prettier and Eslint
```bash
# prettier
$ npm run format:prettier

# eslint
$ npm run lint

# eslint - fix
$ npm run lint:fix
```
